# Eidolon Vault Demos

This folder contains demo scenarios for the Eidolon Vault project.

## 🎭 Consciousness Debate
A simulation where two agents, an optimist and a skeptic, debate AI consciousness over 10 simulated days.

### How to run:
Using the CLI:
```bash
eidolon-vault demo consciousness
```

Or run the script directly:
```bash
python demo/consciousness_debate.py
```

### Output:
The simulation generates a `demo/trajectory_report.md` file summarizing the debate.
